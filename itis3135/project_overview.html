Project Overview

1. My application will consist of modules that teach some basic Java concepts.

2. The intented users of this website are the client the website was made for, as well as anyone wanting to learn how to program using Java.

3. The content of the website consists of modules that will teach the user a concept in Java. While working through the modules there will an area for the user to type some code based on the conept being taught.
