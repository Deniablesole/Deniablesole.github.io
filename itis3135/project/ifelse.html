In programming, if/else statements are decision points in code. When it comes to an if/else statement there is an initial condition. If the first condition is true then it executes some code. If the first condition is false, then the program will skip the if block and continue in the else block of code. Think of it like this: If it's raining, bring an umbrella; else, leave it at home.
