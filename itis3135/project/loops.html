Loops

1 / 3
2 / 3
2 / 3

Above are examples of empty for, while, and do-while loops.

Loops allow us to repeat the same code/action multiple times while a condition evaluates to true. The three types of java loops are the for loop, while loop, and do-while loop. A common mistake that can happen when working is loops is you get an infinite loop. An infinite loop is a loop that will repeat infinitly because the loop will never evaluate to false.

For Loops


In the above for loop we initialize i to be 0. Since we want the loop to run 5 times we put i <= 4. We then increment i by 1 so that it counts as the loop running 1 time. The result of the loop is that it prints to the console the value of i.


While Loops


This is a while loop that does the same action as the for loop above. When using a while loop you must initialize i outside of the loop. I want the loop to run 5 times so we have i <=5. The loop then prints the value of i, then increments i to count as a single run through the loop.

Do-While Loops

This is a do-while loop that has the same output as the while loop above. You still have to initialize i outisde of the loop same as the while loop. This loop prints the value of i, then increments i's value by 1. The loop continues to do this until i = 6. when i = 6, the condition is now false and the program will break out of the loop.

  2. Create a while loop that prints the value of i 12 times. You must initialize i to 0 and you must only use the < sign.
