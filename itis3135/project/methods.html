Above on the first slide is an empty java method. The second slide has a method called calculateSum that adds together two numbers and returns the sum. The third slide has a method called greeting which when called will output the text "Welcome to the grocery store!".

A method is a block of code that is only ran when the method is called. A method takes in one or multiple parameters and are used to perform a certain action. Methods are also referred to as functions. If you find yourself writing code for the same function multiple times, you can create a method to do that task, and then call the method instead of writing the same code again. Methods must be declared within a class and can only be used within that class unless there is some sort of encapsulation, but that is for another time.
